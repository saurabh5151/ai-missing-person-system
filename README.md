# AI Missing Person Detection System 🚨

## 📌 Description
This project is designed to help find missing persons using AI-based face recognition and location tracking. It allows users to search, report, and assist authorities in identifying missing individuals.

---

## 🚀 Features
- 🔍 Upload image to search missing person
- 🤖 AI face matching system
- 📍 Report found person with live location
- 👮 Police dashboard for verification
- 📂 Case creation and management system

---

## 🛠️ Technologies Used
- React.js (Frontend)
- Django (Backend)
- Python (AI Processing)
- REST API
